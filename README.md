# ⚔️ RolePlay Arena — AI Multi-Character Debate Engine

A fully free, single-file AI debate app where multiple characters argue any topic in real-time.

## 🚀 Quick Start

1. Open `index.html` in any browser (no server needed)
2. Enter your free API key (Gemini / Groq / OpenRouter)
3. Pick 2–4 characters
4. Set a topic → Start Debate

## ✨ Features

- **12 preset characters** — Elon Musk, Gandhi, Einstein, Devil's Advocate, Steve Jobs, Narendra Modi, and more
- **Custom characters** — create any personality with custom emoji, color, role, and speaking style
- **Multi-round debates** — 2 to 5 rounds
- **4 debate styles** — Formal, Casual, Heated, Socratic, Comedy Roast
- **4 languages** — English, Hinglish, Hindi, Urdu
- **3 response lengths** — Short, Medium, Long
- **Pause/Resume** mid-debate
- **AI Summary** generated at the end
- **Export** as TXT, JSON, Markdown, or Copy/Share
- **Mobile-first** — perfect on Android/iOS browsers
- **Cyberpunk glassmorphism** UI with per-character color accents

## 🔒 Privacy

Your API key is stored only in your browser's localStorage. It is never sent to any server other than the AI provider you choose.
