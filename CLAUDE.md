# roleplay-arena

## UI/UX Design Rule (permanent)

Use Framer Motion (`framer-motion`, already a dependency) for animations/transitions across this app by default — page transitions, hover/tap states, list enter/exit, modals, drawers. Aim for polished, "pro" UI/UX: smooth easing, consistent motion timing, accessible focus states, responsive layout. Don't add motion where it hurts usability (forms, dense data tables) — judge case by case.

Stack: React 19 + Vite + Tailwind CSS v4 + TypeScript.
